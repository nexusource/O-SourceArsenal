## ❱ 導入【Introduction】
1. Datapacksフォルダに配置。
2. /reload もし必要ならば。
3. 導入に成功するとチャットバー(T)や進捗(L)に通知が出ます。
```Path
%userprofile%\AppData\Roaming\.minecraft\saves\<world>\Datapacks\<here!>
```
